# MemeHelper - генератор мемов с заменой текста

**Задача:** Автоматизация создания текстовых мемов формата с закрашиванием букв.

**Функциональность:**
- Анализ исходной фразы и поиск целевых слов
- Замена части фразы на пользовательские варианты
- Генерация мем-шаблонов с разными комбинациями

**Технологии:** C#, WPF, работа со строками, алгоритмы поиска

**Пример работы:**
Исходная фраза: "Предавать нехорошо, я никогда не предам своих"
Поиск: 
- "я ем своих" → "" → "                    я          е      м своих"
